## Project Summary
Приложение "Умникоины" — это VK Mini App, предназначенное для геймификации посещения учебных и культурных заведений (школ, вузов, библиотек, музеев). Пользователи получают виртуальную валюту (умникоины) за чекины в "местах знаний", отслеживают свой прогресс по ежедневным и еженедельным целям, а также соревнуются в общем рейтинге.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS 4
- **Backend/DB**: Supabase (PostgreSQL)
- **Maps**: MapTiler SDK
- **Integration**: @vkontakte/vk-bridge (данные VK)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **VK Credentials**: App ID (53317461) and Server Key are stored in `.env`

## Architecture
- `src/lib/vk-context.tsx`: Контекст для инициализации и работы с VK Bridge.
- `src/lib/umnicoin-service.ts`: Основная бизнес-логика и управление состоянием.
- `src/lib/friend-service.ts`: Сервис для работы с друзьями и социальными функциями через Supabase.
- `src/components/Map.tsx`: Компонент карты на базе MapTiler.
- `src/app/page.tsx`: Главный экран с табами (Карта, Квесты, Друзья, История, Рейтинг).

## User Preferences
- Используется светлая тема с яркими акцентами (индиго, розовый).
- Интерфейс адаптирован под мобильные устройства (VK Mini App).
- Элементы дизайна вдохновлены современными интерфейсами (закругленные углы, градиенты, блюр).

## Project Guidelines
- Все изменения в логике чекинов должны учитывать кулдаун (1 час).
- Радиус чекина ограничен 200 метрами.
- Использование VK Bridge обязательно для получения данных пользователя.
- Социальные функции (друзья, отслеживание) реализуются через Supabase.
